\frame{
\frametitle{Model world}
\only<1>{
Geometric Brownian Motion (GBM) for wealth $x(t)$,
\be
dx = x\left(\mu dt + \sigma dW\right),
\ee
where:
\bi
\item drift $\mu$ -- deterministic growth;
\item volatility $\sigma$ -- fluctuation amplitude;
\item Wiener increment $dW\sim\mathcal{N}(0,dt)$ -- randomness.\np
\ei
Generalisation of repeated coin toss.\np
Our model world contains $N$ independent trajectories, $x_i(t)$.
}

\frame{
\frametitle{Two growth rates}
\only<1>{
Two ways to quantify the ``average'' wealth growth rate in our model population:\np
{\bf 1. Plutocratic growth rate}\np
Find the population average wealth, $\ave{x}_N$, and compute its ergodic growth rate:
\be
g_\text{plu} \equiv \frac{\D\ln\ave{x}_N}{\Dt}.
\ee
``Plutocratic'' because each individual makes a wealth-weighted contribution to the average.
}
\only<2>{
\bc
\includegraphics[width=9.5cm]{./figs/gplu.pdf}
\ec
}
\only<3>{
Two ways to quantify``average'' wealth growth rate in our model population:\np
{\bf 2. Democratic growth rate}\np
Find the ergodic growth rate of each wealth, $\frac{\D\ln x_i}{\Dt}$, and compute the population average:
\be
g_\text{dem} \equiv \ave{\frac{\D\ln x}{\Dt}}_N.
\ee
``Democratic'' because each individual makes an equal contribution to the average.
}
\only<4>{
\bc
\includegraphics[width=9.5cm]{./figs/gdem.pdf}
\ec
}
}

\frame{
\frametitle{Ergodicity}
For finite $N$, both $g_\text{plu}$ and $g_\text{dem}$ are random variables.\np
We can replace them with model properties as $N\to\infty$:
\bea
&g_\text{plu} = \frac{\D\ln\ave{x}_N}{\Dt} &\rightarrow \gens = \mu;\\
&g_\text{dem} = \ave{\frac{\D\ln x}{\Dt}}_N &\rightarrow \gbar = \mu-\frac{\sigma^2}{2} \text{ (ergodicity)}.
\eea
Good approximations for ``large'' model populations.
}

\frame{
\frametitle{Inequality}
\only<1>{
``A whole menagerie of inequality measures in economics.''\np
\bc
\includegraphics[width=9cm]{./figs/density.pdf}
\ec
%Idea is to summarise a distribution by a single number.\np
}
\only<2>{
\underline{Recipe for inequality metric \`a la EE}\np
\vspace{0.25cm}
1. Note that: if $g_\text{plu} > g_\text{dem}$, then inequality is increasing; and if $g_\text{plu} < g_\text{dem}$, then inequality is decreasing.\np
2. Therefore, define inequality, $J(t)$, as the thing that grows at the {\bf difference} between the two rates:
\be
\frac{dJ}{dt} = g_\text{plu} - g_\text{dem}.
\ee
3. Integrate from time $0$ (equality) to $t$ to get:
\be
J(t) = \ln\ave{x(t)}_N - \ave{\ln x(t)}_N.
\ee
}
\only<3>{
$J$ is the {\bf mean logarithmic deviation} or {\bf Theil's $L$ index}.\np
For a large population, $N\to\infty$, following GBM,
\be
\frac{dJ}{dt} = \gens - \gbar = \frac{\sigma^2}{2} \quad \Rightarrow \quad J(t) = \frac{\sigma^2 t}{2},
\ee
so inequality grows without bound.\np
For general wealth dynamics with ergodicity transformation, $v(x)$, wealth is $v$-normally distributed and
\be
J_v(t) = v\left(\ave{x(t)}_N\right) - \ave{v\left(x(t)\right)}_N.
\ee
}
}
