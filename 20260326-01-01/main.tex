\section{Ergodicities}

\frame{{Ergodicity}
\bi
\item[]<2->
Philosophy: keep the text light. 
\vspace{1cm}
\item[]<3->
Simple ergodicity problem:\newline

\hspace{1.5cm}expected value $\neq$ time average.
\vspace{1cm}
\item[]<4->
But $\approx$150 years of theory behind this.
\ei
}

\frame{{Dynamical systems}

\only<1>{
    Applications often use {\it stochastic} processes. 
    
    Ergodic theory written in {\it deterministic} dynamical systems.

    Consistent because DS describe unobservable states.
}

\uncover<2->{
Dynamical system: $(X, F, \mu, T)$
\vspace{.5cm}}

\uncover<3->{
    \underline{Mental image:} 
    }
\bi
\item[]<4->
$X = $ state space: any thing, $x_i$, you can be.\newline 
\item[]<5->
$F = $ field of sets of $x_i$'s. Coarse-graining: you're special, I know little about you. 
But I may know a set $A\in F$ which you're part of. \newline
\item[]<6->
$\mu$ measure: tells the size of a set, $\mu(A)$.\newline
\item[]<7->
$T = $ measure-preserving time evolution operator.\newline 
Set $A$ today will be $T(A)$ tomorrow, then $T^2(A)$ \etc{}\newline
Size of a set stays the same, $\mu(T^{-1}A)=\mu(A)$. \newline
\ei
\vspace{.5cm}
}

\frame{{Dynamical systems}
\bi
\item<1->
Mathematically, march through time deterministically. 
\item<2->
But not physically visible. 
\item<3->
State $x_i$ deeply hidden, unobservable, ``the true you.'' 
\item<4->
True you unknowable, but we can measure observables,\newline 
$f(x_i)$: how rich are you? \newline
$f(T(x_i))$: how rich tomorrow?
\ei
}

\frame{{Ergodic hierarchy}
Ergodicity means
\bi
\item<1->
Metric transitivity

\uncover<2>{
    ``any differences are transitory.''
    
    For any sets $A, B \in F$, there exists an $n$ such that  $\mu(T^{-n}(A)\cap B)>0$ (mixing).
    \vspace{.3cm}

    Equivalently, ``can't get stuck.''
    
    If $A$ satisfies $T^{-1}(A)=A$, then either $\mu(A)=1$ or $\mu(A)=0$ (only trivial invariant sets).
}
\item<3->[$\to$]
Equality of distributions

\uncover<4>{
    ``Measure $f(x_i)$ for a long time; find values with the same frequency as for $f(x_j)$.''
}
\item<5-7>
[$\to$]
Equality of averages\newline
\uncover<6>{
    ``Averaging over time is equivalent to averaging over the ergodic ensemble.''
    }
\ei
}
