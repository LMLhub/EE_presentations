\frame{{The infamous coin toss}
Two properties: 
\bi
\item
1. Expected value $\ave{x(t+1)}=\frac{1}{2} (r_H x(t) + r_T x(t)= 1.05 x(t).$

By induction, \underline{$\ave{x(t)}=x(0)\times 1.05^t.$}

Per-round growth $g_{\ave{}}=1.05.$
\vspace{1cm}
\item
2. Time-average growth $x(t)=x(0)\times \prod_{\tau=1}^t r_{\tau}=x(0)\times r_H^{n_H}\times r_T^{n_T}$.

per-round growth $g_t=\left(\frac{x(t)}{x(0)}\right)^{1/t}=r_H^{n_H/t}\times r_T^{n_T/t}$

long-time limit $\underline{g=\lim_{t\to\infty} r_H^{n_H/t}\times r_T^{n_T/t}=r_H^{1/2}\times r_T^{1/2}}\approx0.949$ 
\ei
}

\frame{{Dynamical systems}

Dynamical sytem: $(X, F, \mu, T)$

Mental image: 
$X = $ all possible identities $x_i$ (you, me; this dice, that dice...). 
$F = $ coarse-graining: any identity is infinitely fine-grained and particular. Interesting statements can be made about collections $A\in X$ of identities (men, women; six-sided dice, dodecahedral dice...). 
$\mu$ size of a collection, $\mu(A)$
$T = $ time passing. Identities change in time as $T^n(A)$. These changes don't affect the size of the collection, $\mu(T^{-1}A)=\mu(A)$. 

Mathematically, each entity marches through time deterministically. But that's not physically visible. Think of $x_i$ as deeply hidden and unobservable, ``the true you.'' But we can make measurements on $x_i$ ``how tall are you?''
}

\frame{{Ergodic hierarchy}
Ergodicity means
\bi
\item<1->
Metric transitivity
\only<2>{
    ``Any differences are transitory.''
    
    For any sets $A, B \in F$, there exists an $n$ such that  $\mu(T^{-n}(A)\cap B)>0$ (mixing).
    
    Or, equivalently, if $A$ satisfies $T^{-1}(A)=A$, then either $\mu(A)=1$ or $\mu(A)=0$ (only trivial invariant sets).
}
\item<3->[$\to$]
Equality of distributions

\only<3>{
    ``In the long run, measurements in any two systems show the same values equally often.''
}
\item<4->
[$\to$]
Equality of averages
\only<4>{
    ``Averaging over time is equivalent to averaging over the ergodic ensemble.''
    }
\ei
}
