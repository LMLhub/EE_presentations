\frame{{The infamous coin toss}
Two properties: 
\bi
\item<2->
1. Expected value \newline
$\ave{x(t+1)}=\frac{1}{2} \left(r_H x(t) + r_T x(t)\right)= 1.05 x(t).$

By induction, $\ave{x(t)}=x(0)\times 1.05^t.$

Per-round growth \underline{$g_{\ave{}}=1.05.$}
\vspace{1cm}
\item<3->
2. Time-average growth\newline 
$x(t)=x(0) \prod_{\tau=1}^t r_{\tau}=x(0) r_H^{n_H} r_T^{n_T}$.

per-round growth $g=\left(\frac{x(t)}{x(0)}\right)^{1/t}=r_H^{n_H/t} r_T^{n_T/t}$

long-time limit $\lim_{t\to\infty} r_H^{n_H/t} r_T^{n_T/t}=r_H^{1/2} r_T^{1/2}$
\newline
\underline{$g_t\approx0.949$} 
\ei
}

\frame{{The infamous coin toss}
Comments:
\bi
\item<1->
discrete Geometric Brownian Motion (GBM): finance theory. 
\item<2->
multiplicative: evolution, life, investing, learning, growing.
\ei
}

\section{Ergodicities}

\frame{{Ergodicity}
\bi
\item[]<2->
Philosophy: keep the text light. 
\vspace{1cm}
\item[]<3->
Simple ergodicity problem:\newline

\hspace{1.5cm}expected value $\neq$ time average.
\vspace{1cm}
\item[]<4->
But $\approx$150 years of theory behind this.
\ei
}

\frame{{Dynamical systems}

\uncover<2->{
Dynamical sytem: $(X, F, \mu, T)$
\vspace{.5cm}}

\uncover<3->{
    \underline{Mental image:} 
    }
\bi
\item[]<4->
$X = $ all possible identities $x_i$ (you, me; this dice, that dice...).\newline 
\item[]<5->
$F = $ coarse-graining: any $x_i$ infinitely particular. Make statements about collections $A\in F$ of identities (children, Rear Admirals; dice showing a six, pink dice...). \newline
\item[]<6->
$\mu$ size of a collection, $\mu(A)$\newline
\item[]<7->
$T = $ time passing. Identities change according to $T^n(A)$. These changes don't affect the size of the collection, $\mu(T^{-1}A)=\mu(A)$. \newline
\ei
\vspace{.5cm}
}

\frame{{Dynamical systems}
\bi
\item<1->
Mathematically, each entity marches through time deterministically. 
\item<2->
But that's not physically visible. 
\item<3->
Think of $x_i$ as deeply hidden and unobservable, ``the true you.'' 
\item<4->
We don't know the true you, but we can measure observables, ``how tall are you?''
\ei
}

\frame{{Ergodic hierarchy}
Ergodicity means
\bi
\item<1->
Metric transitivity

\uncover<2>{
    ``any differences are transitory.''
    
    For any sets $A, B \in F$, there exists an $n$ such that  $\mu(T^{-n}(A)\cap B)>0$ (mixing).
    \vspace{.3cm}

    Equivalently, ``everything always changes.'
    
    If $A$ satisfies $T^{-1}(A)=A$, then either $\mu(A)=1$ or $\mu(A)=0$ (only trivial invariant sets).
}
\item<3->[$\to$]
Equality of distributions

\uncover<4>{
    ``In the long run, measurements in any two systems show the same values equally often.''
}
\item<5-7>
[$\to$]
Equality of averages
\uncover<6>{
    ``Averaging over time is equivalent to averaging over the ergodic ensemble.''
    }
\ei
}
